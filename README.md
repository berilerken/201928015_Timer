# 201928105_Timer

This is a timer project which has these attributes:

Split screen 3-1-3

Middle is TextView

Sides are Button

Add Timer.

When Click Button 

First click start timer

Second click stop timer

Timer has a limit

Which player over limit, other player wins the game
